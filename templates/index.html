{# ── Security Section ─────────────────────────── #}
Security

Enterprise-Grade Security

Your data and transactions are protected by multiple layers of security.

🔐

bcrypt Password Hashing

All passwords are salted and hashed using bcrypt — never stored in plain text.

🚫

Rate Limiting

Accounts lock for 15 minutes after 5 failed login attempts, preventing brute-force attacks.

⏱️

Session Timeout

Automatic logout after 5 minutes of inactivity to protect your account when you walk away.

💾

Atomic Backups

All data is written atomically with automatic backup files to prevent corruption.

✅

Input Validation

All inputs are validated on the server side — email, phone, password strength, and name format.

📝

Complete Audit Trail

Every operation is logged with timestamps for complete transaction traceability.
